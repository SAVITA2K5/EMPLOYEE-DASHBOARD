# Employee Management Dashboard

A full-stack Employee Management Dashboard built using **React**, **Node.js**, and **Express.js**.  
The application allows users to manage employee records through complete CRUD operations with persistent file-based storage.

---

## Project Overview

This project provides a dashboard to:
- Add new employees
- View all employee records
- Update existing employee details
- Delete employee records

The frontend is developed using **React**, while the backend uses **Node.js with Express.js**.  
Employee data is stored in a **JSON file**, ensuring data persistence across server restarts.

---

## Tech Stack

### Frontend
- React.js
- Axios
- HTML, CSS, JavaScript

### Backend
- Node.js
- Express.js
- CORS
- Body-Parser

### Storage
- JSON file (`employees.json`)
